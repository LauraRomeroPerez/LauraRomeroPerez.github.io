FUENTE GENÉRICA DE FAMILIA

Se refiera a una fuente de texto general, por ejemplo sans-serif o serif. Se aplica con el comando font-family seguido de dos puntos y el nombre de la fuente general.
Son las más utilizadas, ya que si un dispositivo utiliza la página web y no tiene una fuente específica no le va a aparecer.

Por ejemplo, este párrafo esta escrito con sans-serif.

FUENTE ESPECÍFICA DE FAMILIA

Se aplica usando la propiedad font-family seguido de dos puntos y el nombre de la fuente específica entre comillas.

Por ejemplo esto esta escrito con la fuente especifica Calibri.

TAMAÑO DE LETRA CON px

Se puede cambiar el tamaño de la letra utilizando el comando font-size seguido del número de píxeles y de "px".

Por ejemplo, este párrafo tiene una letra de 20px de tamaño.

TAMAÑO DE LETRA CON em

Se puede cambiar el tamaño de la letra con una proporción, usando la propiedad font-size y después poniendo la proporción deseada, seguida de "em".
Esto está bien para manterer la proporción entre enunciados.

Por ejemplo, la letra de este párrafo es 2 veces mayor al tamaño predeterminado.

ESTILO DE FUENTE

Se puede cambiar el estilo de fuente con el comando font-style seguido del estilo con el que se quiere estilizar el texto.

Por ejemplo, este párrafo esta escrito con Italic, lo que hace que este en cursiva.

ALTURA ENTRE LÍNEAS

La altura entre líneas se puede cambiar con el comando line-height, seguido del número de píxeles que quieres que haya entre líneas, seguido de "px".

Por ejemplo, la altura entre líneas de este párrafo es de 45px.

ALINEACIÓN DE TEXTOS

Se pueden alinear los textos usando el comando text-align. Para que este alineado a la derecha se utiliza el comando text-align, seguido de dos puntos y de "right";
para que este alineado en el centro, se utiliza el mismo comando, pero seguido de dos puntos y "centre"

Por ejemplo, este párrafo está alineado en el centro de la página web, mientras que los otros vienen predeterminadamente a la izquierda.

DECORACIÓN DE TEXTO

Se puede decorar el texto de diversas maneras, tachando un texto, subrayando o quitando el subrayado de los enlaces. Esto se hace usando "text-decoration", seguido de la función que quieras.
