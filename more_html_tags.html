Los enlaces se pueden hacer desde diferentes sitios. Para hacer esto se utiliza la etiqueta "a" y el atributo "href" seguido de la url del sitio web entre comillas,
tras cerrar la etiqueta, se escribe el texto que te dirigirá a la página web y que aparecerá subrayado y en azul.

TABLA

Para poner una tabla hace falta usar diferentes etiquetas. La primera es "table" para indicar que quieres insertar una tabla. Para el encabezado de la tabla se utiliza "thead" y para el cuerpo "tbody".
Para poner columnas en el encabezado de la tabla se utiliza "th". Para añadir las filas en el cuerpo se utiliza "td".
Por ejemplo:

APP FECHA CONTENIDO

Instagram 2010 Fotografía y vídeo

COMENTARIO

Los comentarios sirven para que las personas interesadas en el código de la página web puedan entenderse entre ellas, sin que los consumidores lean esos mensajes. Es muy útil para los trabajos en grupo.
Se hace escribiendo el siguiente comando:!--lo que quieras escribir--.
